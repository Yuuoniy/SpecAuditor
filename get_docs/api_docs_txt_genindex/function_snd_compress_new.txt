Function: int snd_compress_new ( struct snd_card * card , int device , int dirn , const char * id , struct snd_compr * compr )

create new compress device

Parameters
  struct snd_card *card
    sound card pointer
  int device
    device number
  int dirn
    device direction, should be of type enum snd_compr_direction
  const char *id
    ID string
  struct snd_compr *compr
    compress device pointer
