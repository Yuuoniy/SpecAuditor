Function: int register_fprobe_syms ( struct fprobe * fp , const char * * syms , int num )

Register fprobe to ftrace by symbols.

Parameters
  struct fprobe *fp
    A fprobe data structure to be registered.
  const char **syms
    An array of target symbols.
  int num
    The number of entries of syms .

Description
  Register fp to the symbols given by syms array. This will be useful if
you are sure the symbols exist in the kernel.
