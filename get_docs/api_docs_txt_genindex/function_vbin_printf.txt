Function: int vbin_printf ( u32 * bin_buf , size_t size , const char * fmt_str , va_list args )

Parse a format string and place args’ binary value in a buffer

Parameters
  u32 *bin_buf
    The buffer to place args’ binary value
  size_t size
    The size of the buffer(by words(32bits), not characters)
  const char *fmt_str
    The format string to use
  va_list args
    Arguments for the format string

Description
  The format follows C99 vsnprintf, except n is ignored, and its argument
is skipped.
