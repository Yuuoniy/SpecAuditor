Macro: alloc_ordered_workqueue_lockdep_map



Parameters
  fmt
    printf format for the name of the workqueue
  flags
    WQ_* flags (only WQ_FREEZABLE and WQ_MEM_RECLAIM are meaningful)
  lockdep_map
    user-defined lockdep_map
  args...
    args for fmt

Description
  Same as alloc_ordered_workqueue but with the a user-define lockdep_map.
Useful for workqueues created with the same purpose and to avoid leaking a
lockdep_map on each workqueue creation.
