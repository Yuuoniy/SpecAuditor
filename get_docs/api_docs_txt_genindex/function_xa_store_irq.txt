Function: void * xa_store_irq ( struct xarray * xa , unsigned long index , void * entry , gfp_t gfp )

Store this entry in the XArray.

Parameters
  struct xarray *xa
    XArray.
  unsigned long index
    Index into array.
  void *entry
    New entry.
  gfp_t gfp
    Memory allocation flags.

Description
  This function is like calling xa_store() except it disables interrupts
while holding the array lock.
