Function: int drm_lspcon_set_mode ( const struct drm_device * dev , struct i2c_adapter * adapter , enum drm_lspcon_mode mode , int time_out )

Change LSPCON’s mode of operation by writing offset (0x80, 0x40)

Parameters
  const struct drm_device *dev
    drm_device to use
  struct i2c_adapter *adapter
    I2C-over-aux adapter
  enum drm_lspcon_mode mode
    required mode of operation
  int time_out
    LSPCON mode change settle timeout
