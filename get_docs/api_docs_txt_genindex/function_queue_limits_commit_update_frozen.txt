Function: int queue_limits_commit_update_frozen ( struct request_queue * q , struct queue_limits * lim )

commit an atomic update of queue limits

Parameters
  struct request_queue *q
    queue to update
  struct queue_limits *lim
    limits to apply

Description
  Apply the limits in lim that were obtained from queue_limits_start_update() and updated with the new values by the caller to q .  Freezes the queue
before the update and unfreezes it after.
