Function: struct rpc_clnt * rpc_clone_client_set_auth ( struct rpc_clnt * clnt , rpc_authflavor_t flavor )

Clone an RPC client structure and set its auth

Parameters
  struct rpc_clnt *clnt
    RPC client whose parameters are copied
  rpc_authflavor_t flavor
    security flavor for new client

Description
  Returns a fresh RPC client or an ERR_PTR.
