Function: unsigned long get_state_synchronize_srcu ( struct srcu_struct * ssp )

Provide an end-of-grace-period cookie

Parameters
  struct srcu_struct *ssp
    srcu_struct to provide cookie for.

Description
  This function returns a cookie that can be passed to poll_state_synchronize_srcu() , which will return true if a full grace
period has elapsed in the meantime.  It is the caller’s responsibility
to make sure that grace period happens, for example, by invoking call_srcu() after return from get_state_synchronize_srcu() .
