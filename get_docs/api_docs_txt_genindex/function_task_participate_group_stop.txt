Function: bool task_participate_group_stop ( struct task_struct * task )

participate in a group stop

Parameters
  struct task_struct *task
    task participating in a group stop

Description
  task has JOBCTL_STOP_PENDING set and is participating in a group stop.
Group stop states are cleared and the group stop count is consumed if JOBCTL_STOP_CONSUME was set.  If the consumption completes the group
stop, the appropriate SIGNAL_* flags are set.
