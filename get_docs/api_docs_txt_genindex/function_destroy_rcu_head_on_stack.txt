Function: void destroy_rcu_head_on_stack ( struct rcu_head * head )

destroy on-stack rcu_head for debugobjects

Parameters
  struct rcu_head *head
    pointer to rcu_head structure to be initialized

Description
  This function informs debugobjects that an on-stack rcu_head structure
is about to go out of scope.  As with init_rcu_head_on_stack() , this
function is not required for rcu_head structures that are statically
defined or that are dynamically allocated on the heap.  Also as with init_rcu_head_on_stack() , this function has no effect for
!CONFIG_DEBUG_OBJECTS_RCU_HEAD kernel builds.
