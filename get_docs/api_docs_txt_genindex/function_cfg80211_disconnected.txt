Function: void cfg80211_disconnected ( struct net_device * dev , u16 reason , const u8 * ie , size_t ie_len , bool locally_generated , gfp_t gfp )

notify cfg80211 that connection was dropped

Parameters
  struct net_device *dev
    network device
  u16 reason
    reason code for the disconnection, set it to 0 if unknown
  const u8 *ie
    information elements of the deauth/disassoc frame (may be NULL )
  size_t ie_len
    length of IEs
  bool locally_generated
    disconnection was requested locally
  gfp_t gfp
    allocation flags

Description
  After it calls this function, the driver should enter an idle state
and not try to connect to any AP any more.
