Function: struct kern_ipc_perm * ipc_obtain_object_check ( struct ipc_ids * ids , int id )

Similar to ipc_obtain_object_idr() but also checks the ipc object sequence number.

Parameters
  struct ipc_ids *ids
    ipc identifier set
  int id
    ipc id to look for

Description
  Call inside the RCU critical section.
The ipc object is not locked on exit.
