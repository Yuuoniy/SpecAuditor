Function: struct kern_ipc_perm * ipc_obtain_object_idr ( struct ipc_ids * ids , int id )

Look for an id in the ipc ids idr and return associated ipc object.

Parameters
  struct ipc_ids *ids
    ipc identifier set
  int id
    ipc id to look for

Description
  Call inside the RCU critical section.
The ipc object is not locked on exit.
