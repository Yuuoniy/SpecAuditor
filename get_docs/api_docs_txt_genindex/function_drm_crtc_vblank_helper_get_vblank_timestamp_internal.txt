Function: bool drm_crtc_vblank_helper_get_vblank_timestamp_internal ( struct drm_crtc * crtc , int * max_error , ktime_t * vblank_time , bool in_vblank_irq , drm_vblank_get_scanout_position_func get_scanout_position )

precise vblank timestamp helper

Parameters
  struct drm_crtc *crtc
    CRTC whose vblank timestamp to retrieve
  int *max_error
    Desired maximum allowable error in timestamps (nanosecs)
On return contains true maximum error of timestamp
  ktime_t *vblank_time
    Pointer to time which should receive the timestamp
  bool in_vblank_irq
    True when called from drm_crtc_handle_vblank() .  Some drivers
need to apply some workarounds for gpu-specific vblank irq quirks
if flag is set.
  drm_vblank_get_scanout_position_func get_scanout_position
    Callback function to retrieve the scanout position. See struct drm_crtc_helper_funcs.get_scanout_position.

Description
  Implements calculation of exact vblank timestamps from given drm_display_mode
timings and current video scanout position of a CRTC.
