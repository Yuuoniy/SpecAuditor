Function: int dvb_vb2_expbuf ( struct dvb_vb2_ctx * ctx , struct dmx_exportbuffer * exp )

Wrapper to vb2_core_expbuf() for Digital TV buffer handling.

Parameters
  struct dvb_vb2_ctx *ctx
    control struct for VB2 handler
  struct dmx_exportbuffer *exp
    struct dmx_exportbuffer passed from userspace in
order to handle DMX_EXPBUF .

Description
  Export a buffer as a file descriptor.
