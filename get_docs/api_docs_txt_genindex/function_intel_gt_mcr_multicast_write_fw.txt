Function: void intel_gt_mcr_multicast_write_fw ( struct intel_gt * gt , i915_mcr_reg_t reg , u32 value )

write a value to all instances of an MCR register

Parameters
  struct intel_gt *gt
    GT structure
  i915_mcr_reg_t reg
    the MCR register to write
  u32 value
    value to write

Description
  Write an MCR register in multicast mode to update all instances.  This
function assumes the caller is already holding any necessary forcewake
domains; use intel_gt_mcr_multicast_write() in cases where forcewake should
be obtained automatically.
