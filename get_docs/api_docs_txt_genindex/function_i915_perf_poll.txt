Function: __poll_t i915_perf_poll ( struct file * file , poll_table * wait )

call poll_wait() with a suitable wait queue for stream

Parameters
  struct file *file
    An i915 perf stream file
  poll_table *wait
    poll() state table

Description
  For handling userspace polling on an i915 perf stream, this ensures poll_wait() gets called with a wait queue that will be woken for new stream
data.
