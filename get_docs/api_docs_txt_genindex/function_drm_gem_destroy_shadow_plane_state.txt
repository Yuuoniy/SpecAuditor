Function: void drm_gem_destroy_shadow_plane_state ( struct drm_plane * plane , struct drm_plane_state * plane_state )

deletes shadow-buffered plane state

Parameters
  struct drm_plane *plane
    the plane
  struct drm_plane_state *plane_state
    the plane state of type struct drm_shadow_plane_state

Description
  This function implements struct drm_plane_funcs.atomic_destroy_state for shadow-buffered planes. It expects that mappings of shadow buffers
have been released already.
