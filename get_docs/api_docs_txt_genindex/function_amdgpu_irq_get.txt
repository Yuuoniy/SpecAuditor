Function: int amdgpu_irq_get ( struct amdgpu_device * adev , struct amdgpu_irq_src * src , unsigned int type )

enable interrupt

Parameters
  struct amdgpu_device *adev
    amdgpu device pointer
  struct amdgpu_irq_src *src
    interrupt source pointer
  unsigned int type
    type of interrupt

Description
  Enables specified type of interrupt on the specified source (all ASICs).
