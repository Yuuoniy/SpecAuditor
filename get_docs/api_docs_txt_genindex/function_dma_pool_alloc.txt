Function: void * dma_pool_alloc ( struct dma_pool * pool , gfp_t mem_flags , dma_addr_t * handle )

get a block of coherent memory

Parameters
  struct dma_pool *pool
    dma pool that will produce the block
  gfp_t mem_flags
    GFP_* bitmask
  dma_addr_t *handle
    pointer to dma address of block
