Function: u32 i915_gem_fence_size ( struct drm_i915_private * i915 , u32 size , unsigned int tiling , unsigned int stride )

required global GTT size for a fence

Parameters
  struct drm_i915_private *i915
    i915 device
  u32 size
    object size
  unsigned int tiling
    tiling mode
  unsigned int stride
    tiling stride

Description
  Return the required global GTT size for a fence (view of a tiled object),
taking into account potential fence register mapping.
