Function: int rio_disc_mport ( struct rio_mport * mport , u32 flags )

Start discovery through a master port

Parameters
  struct rio_mport *mport
    Master port to send transactions
  u32 flags
    discovery control flags

Description
  Starts the discovery process. If we have an active link,
then wait for the signal that enumeration is complete (if wait
is allowed).
When enumeration completion is signaled, start recursive
peer discovery. Returns 0 if discovery succeeds or -EBUSY on failure.
