Function: struct device * tty_port_register_device ( struct tty_port * port , struct tty_driver * driver , unsigned index , struct device * device )

register tty device

Parameters
  struct tty_port *port
    tty_port of the device
  struct tty_driver *driver
    tty_driver for this device
  unsigned index
    index of the tty
  struct device *device
    parent if exists, otherwise NULL

Description
  It is the same as tty_register_device() except the provided port is linked
to a concrete tty specified by index . Use this or tty_port_install() (or
both). Call tty_port_link_device() as a last resort.
