Function: int __generic_file_fsync ( struct file * file , loff_t start , loff_t end , int datasync )

generic fsync implementation for simple filesystems

Parameters
  struct file *file
    file to synchronize
  loff_t start
    start offset in bytes
  loff_t end
    end offset in bytes (inclusive)
  int datasync
    only synchronize essential metadata if true

Description
  This is a generic implementation of the fsync method for simple
filesystems which track all non-inode metadata in the buffers list
hanging off the address_space structure.
