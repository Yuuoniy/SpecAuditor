Function: int scsi_dev_info_list_add_keyed ( int compatible , char * vendor , char * model , char * strflags , blist_flags_t flags , enum scsi_devinfo_key key )

add one dev_info list entry.

Parameters
  int compatible
    if true, null terminate short strings.  Otherwise space pad.
  char *vendor
    vendor string
  char *model
    model (product) string
  char *strflags
    integer string
  blist_flags_t flags
    if strflags NULL, use this flag value
  enum scsi_devinfo_key key
    specify list to use

Description
  Create and add one dev_info entry for vendor , model , strflags or flag in list specified by key . If compatible ,
add to the tail of the list, do not space pad, and set
devinfo->compatible. The scsi_static_device_list entries are
added with compatible 1 and clfags NULL.
