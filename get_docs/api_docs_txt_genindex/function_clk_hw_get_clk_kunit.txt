Function: struct clk * clk_hw_get_clk_kunit ( struct kunit * test , struct clk_hw * hw , const char * con_id )

Test managed clk_hw_get_clk()

Parameters
  struct kunit *test
    The test context
  struct clk_hw *hw
    clk_hw associated with the clk being consumed
  const char *con_id
    connection ID string on device

Description
  Just like clk_hw_get_clk() , except the clk is managed by the test case and
is automatically put with clk_put() after the test case concludes.
