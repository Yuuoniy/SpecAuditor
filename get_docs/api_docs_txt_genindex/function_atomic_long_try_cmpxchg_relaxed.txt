Function: bool atomic_long_try_cmpxchg_relaxed ( atomic_long_t * v , long * old , long new )

atomic compare and exchange with relaxed ordering

Parameters
  atomic_long_t *v
    pointer to atomic_long_t
  long *old
    pointer to long value to compare with
  long new
    long value to assign

Description
  If ( v == old ), atomically updates v to new with relaxed ordering.
Otherwise, v is not modified, old is updated to the current value of v ,
and relaxed ordering is provided.
