Function: void tipc_node_link_up ( struct tipc_node * n , int bearer_id , struct sk_buff_head * xmitq )

handle addition of link

Parameters
  struct tipc_node *n
    target tipc_node
  int bearer_id
    id of the bearer
  struct sk_buff_head *xmitq
    queue for messages to be xmited on

Description
  Link becomes active (alone or shared) or standby, depending on its priority.
