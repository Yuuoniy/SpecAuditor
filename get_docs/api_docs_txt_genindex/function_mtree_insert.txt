Function: int mtree_insert ( struct maple_tree * mt , unsigned long index , void * entry , gfp_t gfp )

Insert an entry at a given index if there is no value.

Parameters
  struct maple_tree *mt
    The maple tree
  unsigned long index
    The index to store the value
  void *entry
    The entry to store
  gfp_t gfp
    The GFP_FLAGS to use for allocations.
