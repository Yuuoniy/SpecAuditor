Function: int crypto_skcipher_setkey ( struct crypto_skcipher * tfm , const u8 * key , unsigned int keylen )

set key for cipher

Parameters
  struct crypto_skcipher *tfm
    cipher handle
  const u8 *key
    buffer holding the key
  unsigned int keylen
    length of the key in bytes

Description
  The caller provided key is set for the skcipher referenced by the cipher
handle.
