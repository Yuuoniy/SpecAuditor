Function: int ida_find_first_range ( struct ida * ida , unsigned int min , unsigned int max )

Get the lowest used ID.

Parameters
  struct ida *ida
    IDA handle.
  unsigned int min
    Lowest ID to get.
  unsigned int max
    Highest ID to get.

Description
  Get the lowest used ID between min and max , inclusive.  The returned
ID will not exceed INT_MAX , even if max is larger.
