Function: void tty_port_link_device ( struct tty_port * port , struct tty_driver * driver , unsigned index )

link tty and tty_port

Parameters
  struct tty_port *port
    tty_port of the device
  struct tty_driver *driver
    tty_driver for this device
  unsigned index
    index of the tty

Description
  Provide the tty layer with a link from a tty (specified by index ) to a
tty_port ( port ). Use this only if neither tty_port_register_device() nor tty_port_install() is used in the driver. If used, this has to be called
before tty_register_driver() .
