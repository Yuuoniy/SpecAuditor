Struct: struct dcp_blob_fmt

DCP BLOB format.

Members
  fmt_version
    Format version, currently being 1 .
  blob_key
    Random AES 128 key which is used to encrypt payload , blob_key itself is encrypted with OTP or UNIQUE device key in
AES-128-ECB mode by DCP.
  nonce
    Random nonce used for payload encryption.
  payload_len
    Length of the plain text payload .
  payload
    The payload itself, encrypted using AES-128-GCM and blob_key ,
GCM auth tag of size DCP_BLOB_AUTHLEN is attached at the end of it.
