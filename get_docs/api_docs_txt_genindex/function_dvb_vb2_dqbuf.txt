Function: int dvb_vb2_dqbuf ( struct dvb_vb2_ctx * ctx , struct dmx_buffer * b )

Wrapper to vb2_core_dqbuf() for Digital TV buffer handling.

Parameters
  struct dvb_vb2_ctx *ctx
    control struct for VB2 handler
  struct dmx_buffer *b
    struct dmx_buffer passed from userspace in
order to handle DMX_DQBUF .

Description
  Dequeue a Digital TV buffer to the userspace
