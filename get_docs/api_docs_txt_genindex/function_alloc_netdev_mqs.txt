Function: struct net_device * alloc_netdev_mqs ( int sizeof_priv , const char * name , unsigned char name_assign_type , void ( * setup ) ( struct net_device * ) , unsigned int txqs , unsigned int rxqs )

allocate network device

Parameters
  int sizeof_priv
    size of private data to allocate space for
  const char *name
    device name format string
  unsigned char name_assign_type
    origin of device name
  void (*setup)(struct net_device *)
    callback to initialize device
  unsigned int txqs
    the number of TX subqueues to allocate
  unsigned int rxqs
    the number of RX subqueues to allocate

Description
  Allocates a struct net_device with private data area for driver use
and performs basic initialization.  Also allocates subqueue structs
for each queue on the device.
