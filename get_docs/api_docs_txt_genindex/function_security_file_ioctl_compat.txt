Function: int security_file_ioctl_compat ( struct file * file , unsigned int cmd , unsigned long arg )

Check if an ioctl is allowed in compat mode

Parameters
  struct file *file
    associated file
  unsigned int cmd
    ioctl cmd
  unsigned long arg
    ioctl arguments

Description
  Compat version of security_file_ioctl() that correctly handles 32-bit
processes running on 64-bit kernels.
