Function: int __mdiobus_c45_read ( struct mii_bus * bus , int addr , int devad , u32 regnum )

Unlocked version of the mdiobus_c45_read function

Parameters
  struct mii_bus *bus
    the mii_bus struct
  int addr
    the phy address
  int devad
    device address to read
  u32 regnum
    register number to read

Description
  Read a MDIO bus register. Caller must hold the mdio bus lock.
