Map the BO into GGTT

Parameters
  struct xe_ggtt *ggtt
    the xe_ggtt where node will be mapped
  struct xe_ggtt_node *node
    the xe_ggtt_node where this BO is mapped
  struct xe_bo *bo
    the xe_bo to be mapped

    Which pat_index to use.
