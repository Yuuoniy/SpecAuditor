Function: void mipi_dsi_dcs_set_page_address_multi ( struct mipi_dsi_multi_context * ctx , u16 start , u16 end )

define the page extent of the frame memory accessed by the host processor

Parameters
  struct mipi_dsi_multi_context *ctx
    Context for multiple DSI transactions
  u16 start
    first page of frame memory
  u16 end
    last page of frame memory

Description
  Like mipi_dsi_dcs_set_page_address() but deals with errors in a way that
makes it convenient to make several calls in a row.
