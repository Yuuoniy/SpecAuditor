Function: struct crypto_ahash * crypto_alloc_ahash ( const char * alg_name , u32 type , u32 mask )

allocate ahash cipher handle

Parameters
  const char *alg_name
    is the cra_name / name or cra_driver_name / driver name of the
ahash cipher
  u32 type
    specifies the type of the cipher
  u32 mask
    specifies the mask for the cipher

Description
  Allocate a cipher handle for an ahash. The returned struct crypto_ahash is the cipher handle that is required for any subsequent
API invocation for that ahash.
