Function: void mac_link_down ( struct phylink_config * config , unsigned int mode , phy_interface_t interface )

notification that the link has gone down

Parameters
  struct phylink_config *config
    a pointer to a struct phylink_config .
  unsigned int mode
    link autonegotiation mode
  phy_interface_t interface
    link typedef phy_interface_t mode

Description
  Notifies the MAC that the link has gone down. This will not be called
unless mac_link_up() has been previously called.
