Function: int of_parse_phandle_with_args_map ( const struct device_node * np , const char * list_name , const char * stem_name , int index , struct of_phandle_args * out_args )

Find a node pointed by phandle in a list and remap it

Parameters
  const struct device_node *np
    pointer to a device tree node containing a list
  const char *list_name
    property name that contains a list
  const char *stem_name
    stem of property names that specify phandles’ arguments count
  int index
    index of a phandle to parse out
  struct of_phandle_args *out_args
    optional pointer to output arguments structure (will be filled)

Description
  This function is useful to parse lists of phandles and their arguments.
Returns 0 on success and fills out_args, on error returns appropriate errno
value. The difference between this function and of_parse_phandle_with_args() is that this API remaps a phandle if the node the phandle points to has
a < stem_name >-map property.
