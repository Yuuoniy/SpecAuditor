Function: bool ata_link_offline ( struct ata_link * link )

test whether the given link is offline

Parameters
  struct ata_link *link
    ATA link to test

Description
  Test whether link is offline.  This is identical to ata_phys_link_offline() when there’s no slave link.  When
there’s a slave link, this function should only be called on
the master link and will return true if both M/S links are
offline.
