Function: unsigned int atapi_eh_request_sense ( struct ata_device * dev , u8 * sense_buf , u8 dfl_sense_key )

perform ATAPI REQUEST_SENSE

Parameters
  struct ata_device *dev
    device to perform REQUEST_SENSE to
  u8 *sense_buf
    result sense data buffer (SCSI_SENSE_BUFFERSIZE bytes long)
  u8 dfl_sense_key
    default sense key to use

Description
  Perform ATAPI REQUEST_SENSE after the device reported CHECK
SENSE.  This function is EH helper.
