Function: void v4l2_async_nf_init ( struct v4l2_async_notifier * notifier , struct v4l2_device * v4l2_dev )

Initialize a notifier.

Parameters
  struct v4l2_async_notifier *notifier
    pointer to struct v4l2_async_notifier
  struct v4l2_device *v4l2_dev
    pointer to struct v4l2_device

Description
  This function initializes the notifier asc_entry . It must be called
before adding a subdevice to a notifier, using one of: v4l2_async_nf_add_fwnode_remote() , v4l2_async_nf_add_fwnode() or v4l2_async_nf_add_i2c() .
