Function: int kobject_uevent_env ( struct kobject * kobj , enum kobject_action action , char * envp_ext [ ] )

send an uevent with environmental data

Parameters
  struct kobject *kobj
    struct kobject that the action is happening to
  enum kobject_action action
    action that is happening
  char *envp_ext[]
    pointer to environmental data

Description
  Returns 0 if kobject_uevent_env() is completed with success or the
corresponding error when it fails.
