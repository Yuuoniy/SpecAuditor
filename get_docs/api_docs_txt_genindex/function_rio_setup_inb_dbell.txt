Function: int rio_setup_inb_dbell ( struct rio_mport * mport , void * dev_id , struct resource * res , void ( * dinb ) ( struct rio_mport * mport , void * dev_id , u16 src , u16 dst , u16 info ) )

bind inbound doorbell callback

Parameters
  struct rio_mport *mport
    RIO master port to bind the doorbell callback
  void *dev_id
    Device specific pointer to pass on event
  struct resource *res
    Doorbell message resource
  void (*dinb) (struct rio_mport * mport, void *dev_id, u16 src, u16 dst, u16 info)
    Callback to execute when doorbell is received

Description
  Adds a doorbell resource/callback pair into a port’s
doorbell event list. Returns 0 if the request has been
satisfied.
