Function: int drm_dp_read_desc ( struct drm_dp_aux * aux , struct drm_dp_desc * desc , bool is_branch )

read sink/branch descriptor from DPCD

Parameters
  struct drm_dp_aux *aux
    DisplayPort AUX channel
  struct drm_dp_desc *desc
    Device descriptor to fill from DPCD
  bool is_branch
    true for branch devices, false for sink devices

Description
  Read DPCD 0x400 (sink) or 0x500 (branch) into desc . Also debug log the
identification.
