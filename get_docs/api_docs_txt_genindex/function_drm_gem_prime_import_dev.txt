Function: struct drm_gem_object * drm_gem_prime_import_dev ( struct drm_device * dev , struct dma_buf * dma_buf , struct device * attach_dev )

core implementation of the import callback

Parameters
  struct drm_device *dev
    drm_device to import into
  struct dma_buf *dma_buf
    dma-buf object to import
  struct device *attach_dev
    struct device to dma_buf attach

Description
  This is the core of drm_gem_prime_import() . It’s designed to be called by
drivers who want to use a different device structure than drm_device.dev for
attaching via dma_buf. This function calls drm_driver.gem_prime_import_sg_table internally.
