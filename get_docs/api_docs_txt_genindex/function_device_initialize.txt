Function: void device_initialize ( struct device * dev )

init device structure.

Parameters
  struct device *dev
    device.

Description
  This prepares the device for use by other layers by initializing
its fields.
It is the first half of device_register() , if called by
that function, though it can also be called separately, so one
may use dev ’s fields. In particular, get_device() / put_device() may be used for reference counting of dev after calling this
function.
