Function: struct iscsi_bus_flash_session * iscsi_create_flashnode_sess ( struct Scsi_Host * shost , int index , struct iscsi_transport * transport , int dd_size )

Add flashnode session entry in sysfs

Parameters
  struct Scsi_Host *shost
    pointer to host data
  int index
    index of flashnode to add in sysfs
  struct iscsi_transport *transport
    pointer to transport data
  int dd_size
    total size to allocate

Description
  Adds a sysfs entry for the flashnode session attributes
