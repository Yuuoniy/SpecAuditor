Function: void i3c_i2c_driver_unregister ( struct i3c_driver * i3cdrv , struct i2c_driver * i2cdrv )

Unregister an i2c and an i3c driver

Parameters
  struct i3c_driver *i3cdrv
    the I3C driver to register
  struct i2c_driver *i2cdrv
    the I2C driver to register

Description
  This function unregisters both i3cdrv and i2cdrv .
Note that when CONFIG_I3C is not enabled, this function only unregisters the i2cdrv .
