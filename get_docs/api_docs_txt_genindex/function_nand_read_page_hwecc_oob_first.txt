Function: int nand_read_page_hwecc_oob_first ( struct nand_chip * chip , uint8_t * buf , int oob_required , int page )

Hardware ECC page read with ECC data read from OOB area

Parameters
  struct nand_chip *chip
    nand chip info structure
  uint8_t *buf
    buffer to store read data
  int oob_required
    caller requires OOB data read to chip->oob_poi
  int page
    page number to read

Description
  Hardware ECC for large page chips, which requires the ECC data to be
extracted from the OOB before the actual data is read.
