Function: int nand_write_page_raw_notsupp ( struct nand_chip * chip , const u8 * buf , int oob_required , int page )

dummy raw page write function

Parameters
  struct nand_chip *chip
    nand chip info structure
  const u8 *buf
    data buffer
  int oob_required
    must write chip->oob_poi to OOB
  int page
    page number to write

Description
  Returns -ENOTSUPP unconditionally.
