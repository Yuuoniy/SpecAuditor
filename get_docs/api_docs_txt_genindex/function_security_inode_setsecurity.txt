Function: int security_inode_setsecurity ( struct inode * inode , const char * name , const void * value , size_t size , int flags )

Set the xattr security label of an inode

Parameters
  struct inode *inode
    inode
  const char *name
    xattr name
  const void *value
    security label
  size_t size
    length of security label
  int flags
    flags

Description
  Set the security label associated with name for inode from the extended
attribute value value . size indicates the size of the value in bytes. flags may be XATTR_CREATE, XATTR_REPLACE, or 0. Note that name is the
remainder of the attribute name after the security. prefix has been removed.
