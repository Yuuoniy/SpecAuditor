Struct: struct sfp_module_caps

sfp module capabilities

Members
  interfaces
    bitmap of interfaces that the module may support
  link_modes
    bitmap of ethtool link modes that the module may support
  may_have_phy
    indicate whether the module may have an ethernet PHY
There is no way to be sure that a module has a PHY as the EEPROM
doesn’t contain this information. When set, this does not mean that
the module definitely has a PHY.
  port
    one of ethtool PORT_* definitions, parsed from the module
EEPROM, or PORT_OTHER if the port type is not known.
