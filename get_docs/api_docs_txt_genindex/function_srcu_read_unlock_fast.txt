Function: void srcu_read_unlock_fast ( struct srcu_struct * ssp , struct srcu_ctr __percpu * scp )

unregister a old reader from an SRCU-protected structure.

Parameters
  struct srcu_struct *ssp
    srcu_struct in which to unregister the old reader.
  struct srcu_ctr __percpu *scp
    return value from corresponding srcu_read_lock_fast() .

Description
  Exit a light-weight SRCU read-side critical section.
