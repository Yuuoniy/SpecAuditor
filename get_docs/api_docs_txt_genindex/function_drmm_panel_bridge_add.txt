Function: struct drm_bridge * drmm_panel_bridge_add ( struct drm_device * drm , struct drm_panel * panel )

Creates a DRM-managed drm_bridge and drm_connector that just calls the appropriate functions from drm_panel .

Parameters
  struct drm_device *drm
    DRM device to tie the bridge lifetime to
  struct drm_panel *panel
    The drm_panel being wrapped.  Must be non-NULL.

Description
  This is the DRM-managed version of drm_panel_bridge_add() which
automatically calls drm_panel_bridge_remove() when dev is cleaned
up.
