Function: struct device_node * of_get_next_child_with_prefix ( const struct device_node * node , struct device_node * prev , const char * prefix )

Find the next child node with prefix

Parameters
  const struct device_node *node
    parent node
  struct device_node *prev
    previous child of the parent node, or NULL to get first
  const char *prefix
    prefix that the node name should have

Description
  This function is like of_get_next_child() , except that it automatically
skips any nodes whose name doesn’t have the given prefix.
