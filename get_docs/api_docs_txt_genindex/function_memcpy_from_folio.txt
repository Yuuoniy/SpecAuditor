Function: void memcpy_from_folio ( char * to , struct folio * folio , size_t offset , size_t len )

Copy a range of bytes from a folio.

Parameters
  char *to
    The memory to copy to.
  struct folio *folio
    The folio to read from.
  size_t offset
    The first byte in the folio to read.
  size_t len
    The number of bytes to copy.
