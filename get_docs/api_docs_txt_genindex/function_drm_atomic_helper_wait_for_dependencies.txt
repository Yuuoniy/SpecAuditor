Function: void drm_atomic_helper_wait_for_dependencies ( struct drm_atomic_state * state )

wait for required preceding commits

Parameters
  struct drm_atomic_state *state
    atomic state object being committed

Description
  This function waits for all preceding commits that touch the same CRTC as state to both be committed to the hardware (as signalled by drm_atomic_helper_commit_hw_done() ) and executed by the hardware (as signalled
by calling drm_crtc_send_vblank_event() on the drm_crtc_state.event ).
