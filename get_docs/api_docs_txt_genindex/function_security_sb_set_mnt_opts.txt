Set the mount options for a filesystem

Parameters
  struct super_block *sb
    filesystem superblock
  void *mnt_opts
    binary mount options
  unsigned long kern_flags
    kernel flags (in)
  unsigned long *set_kern_flags
    kernel flags (out)

Description
