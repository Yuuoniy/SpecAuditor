Function: void mipi_dsi_dcs_set_display_off_multi ( struct mipi_dsi_multi_context * ctx )

send DCS SET_DISPLAY_OFF packet

Parameters
  struct mipi_dsi_multi_context *ctx
    Context for multiple DSI transactions

Description
  Like mipi_dsi_dcs_set_display_off() but deals with errors in a way that
makes it convenient to make several calls in a row.
