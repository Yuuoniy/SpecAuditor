Function: int crypto_aead_setkey ( struct crypto_aead * tfm , const u8 * key , unsigned int keylen )

set key for cipher

Parameters
  struct crypto_aead *tfm
    cipher handle
  const u8 *key
    buffer holding the key
  unsigned int keylen
    length of the key in bytes

Description
  The caller provided key is set for the AEAD referenced by the cipher
handle.
