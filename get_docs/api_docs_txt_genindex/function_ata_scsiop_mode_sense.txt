Function: unsigned int ata_scsiop_mode_sense ( struct ata_device * dev , struct scsi_cmnd * cmd , u8 * rbuf )

Simulate MODE SENSE 6, 10 commands

Parameters
  struct ata_device *dev
    Target device.
  struct scsi_cmnd *cmd
    SCSI command of interest.
  u8 *rbuf
    Response buffer, to which simulated SCSI cmd output is sent.

Description
  Simulate MODE SENSE commands. Assume this is invoked for direct
access devices (e.g. disks) only. There should be no block
descriptor for other device types.
