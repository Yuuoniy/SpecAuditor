Function: void ieee80211_sta_eosp ( struct ieee80211_sta * pubsta )

notify mac80211 about end of SP

Parameters
  struct ieee80211_sta *pubsta
    the station

Description
  When a device transmits frames in a way that it can’t tell
mac80211 in the TX status about the EOSP, it must clear the IEEE80211_TX_STATUS_EOSP bit and call this function instead.
This applies for PS-Poll as well as uAPSD.
