Function: void drm_crtc_vblank_on ( struct drm_crtc * crtc )

enable vblank events on a CRTC

Parameters
  struct drm_crtc *crtc
    CRTC in question

Description
  This functions restores the vblank interrupt state captured with drm_crtc_vblank_off() again and is generally called when enabling crtc . Note
that calls to drm_crtc_vblank_on() and drm_crtc_vblank_off() can be
unbalanced and so can also be unconditionally called in driver load code to
reflect the current hardware state of the crtc.
