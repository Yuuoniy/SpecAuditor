Function: struct scatterlist * sgl_alloc_order ( unsigned long long length , unsigned int order , bool chainable , gfp_t gfp , unsigned int * nent_p )

allocate a scatterlist and its pages

Parameters
  unsigned long long length
    Length in bytes of the scatterlist. Must be at least one
  unsigned int order
    Second argument for alloc_pages()
  bool chainable
    Whether or not to allocate an extra element in the scatterlist
for scatterlist chaining purposes
  gfp_t gfp
    Memory allocation flags
  unsigned int *nent_p
    [out] Number of entries in the scatterlist that have pages
