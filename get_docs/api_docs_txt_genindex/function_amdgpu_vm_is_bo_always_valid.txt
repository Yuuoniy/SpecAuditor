Function: bool amdgpu_vm_is_bo_always_valid ( struct amdgpu_vm * vm , struct amdgpu_bo * bo )

check if the BO is VM always valid

Parameters
  struct amdgpu_vm *vm
    VM to test against.
  struct amdgpu_bo *bo
    BO to be tested.

Description
  Returns true if the BO shares the dma_resv object with the root PD and is
always guaranteed to be valid inside the VM.
