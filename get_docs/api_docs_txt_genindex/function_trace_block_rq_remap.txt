Function: void trace_block_rq_remap ( struct request * rq , dev_t dev , sector_t from )

map request for a block operation request

Parameters
  struct request *rq
    block IO operation request
  dev_t dev
    device for the operation
  sector_t from
    original sector for the operation

Description
  The block operation request rq in q has been remapped.  The block
operation request rq holds the current information and from hold
the original sector.
