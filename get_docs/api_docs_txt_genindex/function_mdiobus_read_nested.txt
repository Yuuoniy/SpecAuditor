Function: int mdiobus_read_nested ( struct mii_bus * bus , int addr , u32 regnum )

Nested version of the mdiobus_read function

Parameters
  struct mii_bus *bus
    the mii_bus struct
  int addr
    the phy address
  u32 regnum
    register number to read

Description
  In case of nested MDIO bus access avoid lockdep false positives by
using mutex_lock_nested() .
