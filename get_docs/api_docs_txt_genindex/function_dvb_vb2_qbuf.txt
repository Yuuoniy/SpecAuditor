Function: int dvb_vb2_qbuf ( struct dvb_vb2_ctx * ctx , struct dmx_buffer * b )

Wrapper to vb2_core_qbuf() for Digital TV buffer handling.

Parameters
  struct dvb_vb2_ctx *ctx
    control struct for VB2 handler
  struct dmx_buffer *b
    struct dmx_buffer passed from userspace in
order to handle DMX_QBUF .

Description
  Queue a Digital TV buffer as requested by userspace
