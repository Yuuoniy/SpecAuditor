Function: void mipi_dsi_turn_on_peripheral_multi ( struct mipi_dsi_multi_context * ctx )

sends a Turn On Peripheral command

Parameters
  struct mipi_dsi_multi_context *ctx
    Context for multiple DSI transactions

Description
  Like mipi_dsi_turn_on_peripheral() but deals with errors in a way that
makes it convenient to make several calls in a row.
