Function: void page_put_link ( void * arg )

Drop the reference to the symlink.

Parameters
  void *arg
    The folio which contains the symlink.

Description
  This is used internally by page_get_link() .  It is exported for use
by filesystems which need to implement a variant of page_get_link() themselves.  Despite the apparent symmetry, filesystems which use page_get_link() do not need to call page_put_link() .
