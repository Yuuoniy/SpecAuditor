Function: int drm_display_info_set_bus_formats ( struct drm_display_info * info , const u32 * formats , unsigned int num_formats )

set the supported bus formats

Parameters
  struct drm_display_info *info
    display info to store bus formats in
  const u32 *formats
    array containing the supported bus formats
  unsigned int num_formats
    the number of entries in the fmts array

Description
  Store the supported bus formats in display info structure.
See MEDIA_BUS_FMT_* definitions in include/uapi/linux/media-bus-format.h for
a full list of available formats.
