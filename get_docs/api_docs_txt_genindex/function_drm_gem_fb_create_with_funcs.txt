Function: struct drm_framebuffer * drm_gem_fb_create_with_funcs ( struct drm_device * dev , struct drm_file * file , const struct drm_format_info * info , const struct drm_mode_fb_cmd2 * mode_cmd , const struct drm_framebuffer_funcs * funcs )

Helper function for the drm_mode_config_funcs.fb_create callback

Parameters
  struct drm_device *dev
    DRM device
  struct drm_file *file
    DRM file that holds the GEM handle(s) backing the framebuffer
  const struct drm_format_info *info
    pixel format information
  const struct drm_mode_fb_cmd2 *mode_cmd
    Metadata from the userspace framebuffer creation request
  const struct drm_framebuffer_funcs *funcs
    vtable to be used for the new framebuffer object

Description
  This function can be used to set drm_framebuffer_funcs for drivers that need
custom framebuffer callbacks. Use drm_gem_fb_create() if you don’t need to
change drm_framebuffer_funcs . The function does buffer size validation.
