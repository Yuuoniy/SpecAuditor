Function: void drm_atomic_helper_async_commit ( struct drm_device * dev , struct drm_atomic_state * state )

commit state asynchronously

Parameters
  struct drm_device *dev
    DRM device
  struct drm_atomic_state *state
    the driver state object

Description
  This function commits a state asynchronously, i.e., not vblank
synchronized. It should be used on a state only when drm_atomic_async_check() succeeds. Async commits are not supposed to swap
the states like normal sync commits, but just do in-place changes on the
current state.
