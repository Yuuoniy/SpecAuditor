Function: void snd_pcm_lib_preallocate_pages ( struct snd_pcm_substream * substream , int type , struct device * data , size_t size , size_t max )

pre-allocation for the given DMA type

Parameters
  struct snd_pcm_substream *substream
    the pcm substream instance
  int type
    DMA type (SNDRV_DMA_TYPE_*)
  struct device *data
    DMA type dependent data
  size_t size
    the requested pre-allocation size in bytes
  size_t max
    the max. allowed pre-allocation size

Description
  Do pre-allocation for the given DMA buffer type.
