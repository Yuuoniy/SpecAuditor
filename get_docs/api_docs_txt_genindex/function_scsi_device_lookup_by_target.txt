Function: struct scsi_device * scsi_device_lookup_by_target ( struct scsi_target * starget , u64 lun )

find a device given the target

Parameters
  struct scsi_target *starget
    SCSI target pointer
  u64 lun
    SCSI Logical Unit Number

Description
  Looks up the scsi_device with the specified lun for a given starget .  The returned scsi_device has an additional reference that
needs to be released with scsi_device_put once you’re done with it.
