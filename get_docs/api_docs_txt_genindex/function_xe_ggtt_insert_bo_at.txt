Function: int xe_ggtt_insert_bo_at ( struct xe_ggtt * ggtt , struct xe_bo * bo , u64 start , u64 end , struct drm_exec * exec )

Insert BO at a specific GGTT space

Parameters
  struct xe_ggtt *ggtt
    the xe_ggtt where bo will be inserted
  struct xe_bo *bo
    the xe_bo to be inserted
  u64 start
    address where it will be inserted
  u64 end
    end of the range where it will be inserted
  struct drm_exec *exec
    The drm_exec transaction to use for exhaustive eviction.
