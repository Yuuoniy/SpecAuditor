Function: bool sg_miter_next ( struct sg_mapping_iter * miter )

proceed mapping iterator to the next mapping

Parameters
  struct sg_mapping_iter *miter
    sg mapping iter to proceed

Description
  Proceeds miter to the next mapping. miter should have been started
using sg_miter_start() .  On successful return, miter->page , miter->addr and miter->length point to the current mapping.
