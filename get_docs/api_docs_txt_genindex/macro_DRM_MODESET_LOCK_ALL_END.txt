Macro: DRM_MODESET_LOCK_ALL_END



Parameters
  dev
    drm device
  ctx
    local modeset acquire context, will be dereferenced
  ret
    local ret/err/etc variable to track error status

Description
  The other side of DRM_MODESET_LOCK_ALL_BEGIN() . It will bounce back to BEGIN
if ret is -EDEADLK.
