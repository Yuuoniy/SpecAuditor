Function: int v4l2_fwnode_device_parse ( struct device * dev , struct v4l2_fwnode_device_properties * props )

parse fwnode device properties

Parameters
  struct device *dev
    pointer to struct device
  struct v4l2_fwnode_device_properties *props
    pointer to struct v4l2_fwnode_device_properties where to store the
parsed properties values

Description
  This function parses and validates the V4L2 fwnode device properties from the
firmware interface, and fills the struct v4l2_fwnode_device_properties
provided by the caller.
