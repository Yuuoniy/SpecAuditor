Function: unsigned long gen_pool_alloc_algo_owner ( struct gen_pool * pool , size_t size , genpool_algo_t algo , void * data , void * * owner )

allocate special memory from the pool

Parameters
  struct gen_pool *pool
    pool to allocate from
  size_t size
    number of bytes to allocate from the pool
  genpool_algo_t algo
    algorithm passed from caller
  void *data
    data passed to algorithm
  void **owner
    optionally retrieve the chunk owner

Description
  Allocate the requested number of bytes from the specified pool.
Uses the pool allocation function (with first-fit algorithm by default).
Can not be used in NMI handler on architectures without
NMI-safe cmpxchg implementation.
