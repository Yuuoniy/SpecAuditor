Function: struct sg_table * drm_gem_dma_object_get_sg_table ( struct drm_gem_object * obj )

GEM object function for drm_gem_dma_get_sg_table()

Parameters
  struct drm_gem_object *obj
    GEM object

Description
  This function wraps drm_gem_dma_get_sg_table() . Drivers that employ the DMA helpers should
