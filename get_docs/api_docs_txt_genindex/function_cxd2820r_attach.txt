Function: struct dvb_frontend * cxd2820r_attach ( const struct cxd2820r_config * config , struct i2c_adapter * i2c , int * gpio_chip_base )

Attach a cxd2820r demod

Parameters
  const struct cxd2820r_config *config
    pointer to struct cxd2820r_config with demod configuration.
  struct i2c_adapter *i2c
    i2c adapter to use.
  int *gpio_chip_base
    if zero, disables GPIO setting. Otherwise, if
CONFIG_GPIOLIB is set dynamically allocate
gpio base; if is not set, use its value to
setup the GPIO pins.
