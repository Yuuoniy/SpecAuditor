Function: int gnet_stats_copy_queue ( struct gnet_dump * d , struct gnet_stats_queue __percpu * cpu_q , struct gnet_stats_queue * q , __u32 qlen )

copy queue statistics into statistics TLV

Parameters
  struct gnet_dump *d
    dumping handle
  struct gnet_stats_queue __percpu *cpu_q
    per cpu queue statistics
  struct gnet_stats_queue *q
    queue statistics
  __u32 qlen
    queue length statistics

Description
  Appends the queue statistics to the top level TLV created by gnet_stats_start_copy() . Using per cpu queue statistics if
they are available.
