Function: void mipi_dsi_dcs_write_buffer_multi ( struct mipi_dsi_multi_context * ctx , const void * data , size_t len )

mipi_dsi_dcs_write_buffer_chatty() w/ accum_err

Parameters
  struct mipi_dsi_multi_context *ctx
    Context for multiple DSI transactions
  const void *data
    buffer containing data to be transmitted
  size_t len
    size of transmission buffer

Description
  Like mipi_dsi_dcs_write_buffer_chatty() but deals with errors in a way that
makes it convenient to make several calls in a row.
