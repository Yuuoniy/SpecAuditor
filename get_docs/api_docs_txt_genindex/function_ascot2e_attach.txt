Attach an ascot2e tuner

Parameters
  struct dvb_frontend *fe
    frontend to be attached
  const struct ascot2e_config *config
    pointer to struct ascot2e_config with tuner configuration.
  struct i2c_adapter *i2c
    i2c adapter to use.
