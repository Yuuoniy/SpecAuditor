Function: int security_inode_rename ( struct inode * old_dir , struct dentry * old_dentry , struct inode * new_dir , struct dentry * new_dentry , unsigned int flags )

Check if renaming a file is allowed

Parameters
  struct inode *old_dir
    parent directory of the old file
  struct dentry *old_dentry
    the old file
  struct inode *new_dir
    parent directory of the new file
  struct dentry *new_dentry
    the new file
  unsigned int flags
    flags

Description
  Check for permission to rename a file or directory.
