Function: int i3c_device_do_priv_xfers ( struct i3c_device * dev , struct i3c_priv_xfer * xfers , int nxfers )

do I3C SDR private transfers directed to a specific device

Parameters
  struct i3c_device *dev
    device with which the transfers should be done
  struct i3c_priv_xfer *xfers
    array of transfers
  int nxfers
    number of transfers

Description
  Initiate one or several private SDR transfers with dev .
