Function: int kthread_park ( struct task_struct * k )

park a thread created by kthread_create() .

Parameters
  struct task_struct *k
    thread created by kthread_create() .

Description
  Sets kthread_should_park() for k to return true, wakes it, and
waits for it to return. This can also be called after kthread_create() instead of calling wake_up_process() : the thread will park without
calling threadfn() .
