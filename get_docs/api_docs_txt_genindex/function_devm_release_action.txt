Function: void devm_release_action ( struct device * dev , void ( * action ) ( void * ) , void * data )

release previously added custom action

Parameters
  struct device *dev
    Device that owns the action
  void (*action)(void *)
    Function implementing the action
  void *data
    Pointer to data passed to action implementation

Description
  Releases and removes instance of action previously added by devm_add_action() .  Both action and data should match one of the
existing entries.
