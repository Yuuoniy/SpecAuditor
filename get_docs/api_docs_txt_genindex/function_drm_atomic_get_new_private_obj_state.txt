Function: struct drm_private_state * drm_atomic_get_new_private_obj_state ( const struct drm_atomic_state * state , struct drm_private_obj * obj )



Parameters
  const struct drm_atomic_state *state
    global atomic state object
  struct drm_private_obj *obj
    private_obj to grab

Description
  This function returns the new private object state for the given private_obj,
or NULL if the private_obj is not part of the global atomic state.
