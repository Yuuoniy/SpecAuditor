Function: struct xbc_node * xbc_node_get_next ( struct xbc_node * node )

Get the next sibling XBC node

Parameters
  struct xbc_node *node
    An XBC node.

Description
  Return the NEXT sibling node of node . If the node has no next sibling,
return NULL. Note that even if this returns NULL, it doesn’t mean node has no siblings. (You also has to check whether the parent’s child node
is node or not.)
