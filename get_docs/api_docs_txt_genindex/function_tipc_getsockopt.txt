Function: int tipc_getsockopt ( struct socket * sock , int lvl , int opt , char __user * ov , int __user * ol )

get socket option

Parameters
  struct socket *sock
    socket structure
  int lvl
    option level
  int opt
    option identifier
  char __user *ov
    receptacle for option value
  int __user *ol
    receptacle for length of option value

Description
  For stream sockets only, returns 0 length result for all IPPROTO_TCP options
(to ease compatibility).
