Function: int filemap_fdatawait_range ( struct address_space * mapping , loff_t start_byte , loff_t end_byte )

wait for writeback to complete

Parameters
  struct address_space *mapping
    address space structure to wait for
  loff_t start_byte
    offset in bytes where the range starts
  loff_t end_byte
    offset in bytes where the range ends (inclusive)

Description
  Walk the list of under-writeback pages of the given address space
in the given range and wait for all of them.  Check error status of
the address space and return it.
