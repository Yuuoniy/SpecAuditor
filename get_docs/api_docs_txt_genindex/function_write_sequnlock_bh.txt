Function: void write_sequnlock_bh ( seqlock_t * sl )

end a softirqs-disabled seqlock_t write section

Parameters
  seqlock_t *sl
    Pointer to seqlock_t

Description
  write_sequnlock_bh closes the serialized, non-preemptible, and
softirqs-disabled, seqlock_t write side critical section opened with write_seqlock_bh() .
