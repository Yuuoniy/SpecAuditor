Function: void media_gobj_create ( struct media_device * mdev , enum media_gobj_type type , struct media_gobj * gobj )

Initialize a graph object

Parameters
  struct media_device *mdev
    Pointer to the media_device that contains the object
  enum media_gobj_type type
    Type of the object
  struct media_gobj *gobj
    Pointer to the struct media_gobj graph object

Description
  This routine initializes the embedded struct media_gobj inside a
media graph object. It is called automatically if media_*_create function calls are used. However, if the object (entity, link, pad,
interface) is embedded on some other object, this function should be
called before registering the object at the media controller.
