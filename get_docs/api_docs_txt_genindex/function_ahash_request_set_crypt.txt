Function: void ahash_request_set_crypt ( struct ahash_request * req , struct scatterlist * src , u8 * result , unsigned int nbytes )

set data buffers

Parameters
  struct ahash_request *req
    ahash_request handle to be updated
  struct scatterlist *src
    source scatter/gather list
  u8 *result
    buffer that is filled with the message digest -- the caller must
ensure that the buffer has sufficient space by, for example, calling crypto_ahash_digestsize()
  unsigned int nbytes
    number of bytes to process from the source scatter/gather list

Description
  By using this call, the caller references the source scatter/gather list.
The source scatter/gather list points to the data the message digest is to
be calculated for.
