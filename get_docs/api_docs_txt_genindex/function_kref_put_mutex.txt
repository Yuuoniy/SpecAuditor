Function: int kref_put_mutex ( struct kref * kref , void ( * release ) ( struct kref * kref ) , struct mutex * mutex )

Decrement refcount for object

Parameters
  struct kref *kref
    Object
  void (*release)(struct kref *kref)
    Pointer to the function that will clean up the object when the
last reference to the object is released.
  struct mutex *mutex
    Mutex which protects the release function.

Description
  This variant of kref_lock() calls the release function with the mutex held.  The release function will release the mutex.
