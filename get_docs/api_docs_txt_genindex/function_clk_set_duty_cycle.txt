Function: int clk_set_duty_cycle ( struct clk * clk , unsigned int num , unsigned int den )

adjust the duty cycle ratio of a clock signal

Parameters
  struct clk *clk
    clock signal source
  unsigned int num
    numerator of the duty cycle ratio to be applied
  unsigned int den
    denominator of the duty cycle ratio to be applied

Description
  Adjust the duty cycle of a clock signal by the specified ratio. Returns 0 on
success, -EERROR otherwise.
