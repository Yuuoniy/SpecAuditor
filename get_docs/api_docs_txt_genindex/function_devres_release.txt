Function: int devres_release ( struct device * dev , dr_release_t release , dr_match_t match , void * match_data )

Find a device resource and destroy it, calling release

Parameters
  struct device *dev
    Device to find resource from
  dr_release_t release
    Look for resources associated with this release function
  dr_match_t match
    Match function (optional)
  void *match_data
    Data for the match function

Description
  Find the latest devres of dev associated with release and for
which match returns 1.  If match is NULL, it’s considered to
match all.  If found, the resource is removed atomically, the
release function called and the resource freed.
