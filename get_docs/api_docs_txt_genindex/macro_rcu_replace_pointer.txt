Macro: rcu_replace_pointer



Parameters
  rcu_ptr
    RCU pointer, whose old value is returned
  ptr
    regular pointer
  c
    the lockdep conditions under which the dereference will take place

Description
  Perform a replacement, where rcu_ptr is an RCU-annotated
pointer and c is the lockdep argument that is passed to the rcu_dereference_protected() call used to read that pointer.  The old
value of rcu_ptr is returned, and rcu_ptr is set to ptr .
