Function: int snd_ctl_rename_id ( struct snd_card * card , struct snd_ctl_elem_id * src_id , struct snd_ctl_elem_id * dst_id )

replace the id of a control on the card

Parameters
  struct snd_card *card
    the card instance
  struct snd_ctl_elem_id *src_id
    the old id
  struct snd_ctl_elem_id *dst_id
    the new id

Description
  Finds the control with the old id from the card, and replaces the
id with the new one.
