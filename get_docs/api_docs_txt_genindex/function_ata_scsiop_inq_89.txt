Function: unsigned int ata_scsiop_inq_89 ( struct ata_device * dev , struct scsi_cmnd * cmd , u8 * rbuf )

Simulate INQUIRY VPD page 89, ATA info

Parameters
  struct ata_device *dev
    Target device.
  struct scsi_cmnd *cmd
    SCSI command of interest.
  u8 *rbuf
    Response buffer, to which simulated SCSI cmd output is sent.

Description
  Yields SAT-specified ATA VPD page.
