Function: struct reset_control * of_reset_control_get_optional_exclusive ( struct device_node * node , const char * id )

Lookup and obtain an optional exclusive reference to a reset controller.

Parameters
  struct device_node *node
    device to be reset by the controller
  const char *id
    reset line name

Description
  Optional variant of of_reset_control_get_exclusive() . If the requested reset
is not specified in the device tree, this function returns NULL instead of
an error.
