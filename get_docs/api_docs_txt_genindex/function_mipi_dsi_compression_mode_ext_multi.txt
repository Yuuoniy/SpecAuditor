Function: void mipi_dsi_compression_mode_ext_multi ( struct mipi_dsi_multi_context * ctx , bool enable , enum mipi_dsi_compression_algo algo , unsigned int pps_selector )

enable/disable DSC on the peripheral

Parameters
  struct mipi_dsi_multi_context *ctx
    Context for multiple DSI transactions
  bool enable
    Whether to enable or disable the DSC
  enum mipi_dsi_compression_algo algo
    Selected compression algorithm
  unsigned int pps_selector
    Select PPS from the table of pre-stored or uploaded PPS entries

Description
  Like mipi_dsi_compression_mode_ext() but deals with errors in a way that
makes it convenient to make several calls in a row.
