Function: int ata_set_mode ( struct ata_link * link , struct ata_device * * r_failed_dev )

Program timings and issue SET FEATURES - XFER

Parameters
  struct ata_link *link
    link on which timings will be programmed
  struct ata_device **r_failed_dev
    out parameter for failed device

Description
  Standard implementation of the function used to tune and set
ATA device disk transfer mode (PIO3, UDMA6, etc.).  If ata_dev_set_mode() fails, pointer to the failing device is
returned in r_failed_dev .
