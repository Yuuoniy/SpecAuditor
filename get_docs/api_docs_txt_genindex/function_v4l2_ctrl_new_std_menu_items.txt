Function: struct v4l2_ctrl * v4l2_ctrl_new_std_menu_items ( struct v4l2_ctrl_handler * hdl , const struct v4l2_ctrl_ops * ops , u32 id , u8 max , u64 mask , u8 def , const char * const * qmenu )

Create a new standard V4L2 menu control with driver specific menu.

Parameters
  struct v4l2_ctrl_handler *hdl
    The control handler.
  const struct v4l2_ctrl_ops *ops
    The control ops.
  u32 id
    The control ID.
  u8 max
    The control’s maximum value.
  u64 mask
    The control’s skip mask for menu controls. This makes it
easy to skip menu items that are not valid. If bit X is set,
then menu item X is skipped. Of course, this only works for
menus with <= 64 menu items. There are no menus that come
close to that number, so this is OK. Should we ever need more,
then this will have to be extended to a bit array.
  u8 def
    The control’s default value.
  const char * const *qmenu
    The new menu.

Description
  Same as v4l2_ctrl_new_std_menu() , but qmenu will be the driver specific
menu of this control.
