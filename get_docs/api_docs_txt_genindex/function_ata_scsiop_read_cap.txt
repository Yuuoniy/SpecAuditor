Function: unsigned int ata_scsiop_read_cap ( struct ata_device * dev , struct scsi_cmnd * cmd , u8 * rbuf )

Simulate READ CAPACITY[ 16] commands

Parameters
  struct ata_device *dev
    Target device.
  struct scsi_cmnd *cmd
    SCSI command of interest.
  u8 *rbuf
    Response buffer, to which simulated SCSI cmd output is sent.

Description
  Simulate READ CAPACITY commands.
