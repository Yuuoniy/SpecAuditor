Function: int crypto_cipher_setkey ( struct crypto_cipher * tfm , const u8 * key , unsigned int keylen )

set key for cipher

Parameters
  struct crypto_cipher *tfm
    cipher handle
  const u8 *key
    buffer holding the key
  unsigned int keylen
    length of the key in bytes

Description
  The caller provided key is set for the single block cipher referenced by the
cipher handle.
