Function: int __phy_clear_bits_mmd ( struct phy_device * phydev , int devad , u32 regnum , u16 val )

Convenience function for clearing bits in a register on MMD

Parameters
  struct phy_device *phydev
    the phy_device struct
  int devad
    the MMD containing register to modify
  u32 regnum
    register number to modify
  u16 val
    bits to clear

Description
  The caller must have taken the MDIO bus lock.
