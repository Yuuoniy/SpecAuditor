Function: struct dma_fence * dma_resv_iter_next_unlocked ( struct dma_resv_iter * cursor )

next fence in an unlocked dma_resv obj.

Parameters
  struct dma_resv_iter *cursor
    the cursor with the current position

Description
  Beware that the iterator can be restarted.  Code which accumulates statistics
or similar needs to check for this with dma_resv_iter_is_restarted() . For
this reason prefer the locked dma_resv_iter_next() whenever possible.
