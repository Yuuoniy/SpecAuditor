Function: int mtree_store_range ( struct maple_tree * mt , unsigned long index , unsigned long last , void * entry , gfp_t gfp )

Store an entry at a given range.

Parameters
  struct maple_tree *mt
    The maple tree
  unsigned long index
    The start of the range
  unsigned long last
    The end of the range
  void *entry
    The entry to store
  gfp_t gfp
    The GFP_FLAGS to use for allocations
