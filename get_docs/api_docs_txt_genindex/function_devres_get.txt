Function: void * devres_get ( struct device * dev , void * new_res , dr_match_t match , void * match_data )

Find devres, if non-existent, add one atomically

Parameters
  struct device *dev
    Device to lookup or add devres for
  void *new_res
    Pointer to new initialized devres to add if not found
  dr_match_t match
    Match function (optional)
  void *match_data
    Data for the match function

Description
  Find the latest devres of dev which has the same release function
as new_res and for which match return 1.  If found, new_res is
freed; otherwise, new_res is added atomically.
