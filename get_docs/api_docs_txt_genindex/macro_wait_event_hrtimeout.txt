Macro: wait_event_hrtimeout



Parameters
  wq_head
    the waitqueue to wait on
  condition
    a C expression for the event to wait for
  timeout
    timeout, as a ktime_t

Description
  The process is put to sleep (TASK_UNINTERRUPTIBLE) until the condition evaluates to true or a signal is received.
The condition is checked each time the waitqueue wq_head is woken up.
