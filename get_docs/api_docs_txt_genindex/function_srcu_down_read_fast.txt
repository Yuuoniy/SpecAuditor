Function: struct srcu_ctr __percpu * srcu_down_read_fast ( struct srcu_struct * ssp )

register a new reader for an SRCU-protected structure.

Parameters
  struct srcu_struct *ssp
    srcu_struct in which to register the new reader.

Description
  Enter a semaphore-like SRCU read-side critical section, but for
a light-weight smp_mb() -free reader.  See srcu_read_lock_fast() and srcu_down_read() for more information.
