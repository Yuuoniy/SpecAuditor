Function: int add_disk_fwnode ( struct device * parent , struct gendisk * disk , const struct attribute_group * * groups , struct fwnode_handle * fwnode )

add disk information to kernel list with fwnode

Parameters
  struct device *parent
    parent device for the disk
  struct gendisk *disk
    per-device partitioning information
  const struct attribute_group **groups
    Additional per-device sysfs groups
  struct fwnode_handle *fwnode
    attached disk fwnode

Description
  This function registers the partitioning information in disk with the kernel. Also attach a fwnode to the disk device.
