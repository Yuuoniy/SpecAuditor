Resource-managed gpiod_get_array_optional()

Parameters
  struct device *dev
    GPIO consumer
  const char *con_id
    function within the GPIO consumer
  enum gpiod_flags flags
    optional GPIO initialization flags

Description
  Managed gpiod_get_array_optional() . GPIO descriptors returned from this

return values.
