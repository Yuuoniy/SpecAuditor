Function: int snd_pcm_new_stream ( struct snd_pcm * pcm , int stream , int substream_count )

create a new PCM stream

Parameters
  struct snd_pcm *pcm
    the pcm instance
  int stream
    the stream direction, SNDRV_PCM_STREAM_XXX
  int substream_count
    the number of substreams

Description
  Creates a new stream for the pcm.
The corresponding stream on the pcm must have been empty before
calling this, i.e. zero must be given to the argument of snd_pcm_new() .
