Function: int amdgpu_bo_set_tiling_flags ( struct amdgpu_bo * bo , u64 tiling_flags )

set tiling flags

Parameters
  struct amdgpu_bo *bo
    amdgpu_bo buffer object
  u64 tiling_flags
    new flags

Description
  Sets buffer object’s tiling flags with the new one. Used by GEM ioctl or
kernel driver to set the tiling flags on a buffer.
