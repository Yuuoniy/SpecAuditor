Function: void bitmap_write ( unsigned long * map , unsigned long value , unsigned long start , unsigned long nbits )

write n-bit value within a memory region

Parameters
  unsigned long *map
    address to the bitmap memory region
  unsigned long value
    value to write, clamped to nbits
  unsigned long start
    bit offset of the n-bit value
  unsigned long nbits
    size of value in bits, nonzero, up to BITS_PER_LONG.

Description
  bitmap_write() behaves as-if implemented as nbits calls of __assign_bit() ,
i.e. bits beyond nbits are ignored:
