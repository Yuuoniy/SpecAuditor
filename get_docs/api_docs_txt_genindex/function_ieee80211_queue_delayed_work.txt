Function: void ieee80211_queue_delayed_work ( struct ieee80211_hw * hw , struct delayed_work * dwork , unsigned long delay )

add work onto the mac80211 workqueue

Parameters
  struct ieee80211_hw *hw
    the hardware struct for the interface we are adding work for
  struct delayed_work *dwork
    delayable work to queue onto the mac80211 workqueue
  unsigned long delay
    number of jiffies to wait before queueing

Description
  Drivers and mac80211 use this to queue delayed work onto the mac80211
workqueue.
