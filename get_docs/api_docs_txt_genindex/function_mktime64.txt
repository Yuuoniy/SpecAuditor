Function: time64_t mktime64 ( const unsigned int year0 , const unsigned int mon0 , const unsigned int day , const unsigned int hour , const unsigned int min , const unsigned int sec )

Converts date to seconds.

Parameters
  const unsigned int year0
    year to convert
  const unsigned int mon0
    month to convert
  const unsigned int day
    day to convert
  const unsigned int hour
    hour to convert
  const unsigned int min
    minute to convert
  const unsigned int sec
    second to convert

Description
  Converts Gregorian date to seconds since 1970-01-01 00:00:00.
Assumes input in normal date format, i.e. 1980-12-31 23:59:59
=> year=1980, mon=12, day=31, hour=23, min=59, sec=59.
