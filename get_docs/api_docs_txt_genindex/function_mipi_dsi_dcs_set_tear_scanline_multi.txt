Function: void mipi_dsi_dcs_set_tear_scanline_multi ( struct mipi_dsi_multi_context * ctx , u16 scanline )

set the scanline to use as trigger for the Tearing Effect output signal of the display module

Parameters
  struct mipi_dsi_multi_context *ctx
    Context for multiple DSI transactions
  u16 scanline
    scanline to use as trigger

Description
  Like mipi_dsi_dcs_set_tear_scanline() but deals with errors in a way that
makes it convenient to make several calls in a row.
