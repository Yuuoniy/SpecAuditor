Function: int remap_pfn_range ( struct vm_area_struct * vma , unsigned long addr , unsigned long pfn , unsigned long size , pgprot_t prot )

remap kernel memory to userspace

Parameters
  struct vm_area_struct *vma
    user vma to map to
  unsigned long addr
    target page aligned user address to start at
  unsigned long pfn
    page frame number of kernel physical memory address
  unsigned long size
    size of mapping area
  pgprot_t prot
    page protection flags for this mapping
