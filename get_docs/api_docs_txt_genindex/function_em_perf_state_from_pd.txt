Function: struct em_perf_state * em_perf_state_from_pd ( struct em_perf_domain * pd )

Get the performance states table of perf. domain

Parameters
  struct em_perf_domain *pd
    performance domain for which this must be done

Description
  To use this function the rcu_read_lock() should be hold. After the usage
of the performance states table is finished, the rcu_read_unlock() should
be called.
