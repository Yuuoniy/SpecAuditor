Function: const char * xattr_full_name ( const struct xattr_handler * handler , const char * name )

Compute full attribute name from suffix

Parameters
  const struct xattr_handler *handler
    handler of the xattr_handler operation
  const char *name
    name passed to the xattr_handler operation

Description
  The get and set xattr handler operations are called with the remainder of
the attribute name after skipping the handler’s prefix: for example, “foo”
is passed to the get operation of a handler with prefix “user.” to get
attribute “user.foo”.  The full name is still “there” in the name though.
