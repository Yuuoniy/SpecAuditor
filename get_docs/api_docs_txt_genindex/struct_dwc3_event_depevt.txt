Device Endpoint Events

Members

    The event we have:
0x00    - Reserved
0x01    - XferComplete
0x02    - XferInProgress
0x03    - XferNotReady
0x04    - RxTxFifoEvt (IN->Underrun, OUT->Overrun)
0x05    - Reserved
0x06    - StreamEvt
0x07    - EPCmdCmplt
