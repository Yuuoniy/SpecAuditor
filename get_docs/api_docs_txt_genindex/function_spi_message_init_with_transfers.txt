Function: void spi_message_init_with_transfers ( struct spi_message * m , struct spi_transfer * xfers , unsigned int num_xfers )

Initialize spi_message and append transfers

Parameters
  struct spi_message *m
    spi_message to be initialized
  struct spi_transfer *xfers
    An array of SPI transfers
  unsigned int num_xfers
    Number of items in the xfer array

Description
  This function initializes the given spi_message and adds each spi_transfer in
the given array to the message.
