Function: bool __init_memblock memblock_validate_numa_coverage ( unsigned long threshold_bytes )

check if amount of memory with no node ID assigned is less than a threshold

Parameters
  unsigned long threshold_bytes
    maximal memory size that can have unassigned node
ID (in bytes).

Description
  A buggy firmware may report memory that does not belong to any node.
Check if amount of such memory is below threshold_bytes .
