Function: int irq_domain_push_irq ( struct irq_domain * domain , int virq , void * arg )

Push a domain in to the top of a hierarchy.

Parameters
  struct irq_domain *domain
    Domain to push.
  int virq
    Irq to push the domain in to.
  void *arg
    Passed to the irq_domain_ops alloc() function.

Description
  For an already existing irqdomain hierarchy, as might be obtained
via a call to pci_enable_msix() , add an additional domain to the
head of the processing chain.  Must be called before request_irq() has been called.
