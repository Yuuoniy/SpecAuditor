Function: int blk_rq_map_user_iov ( struct request_queue * q , struct request * rq , struct rq_map_data * map_data , const struct iov_iter * iter , gfp_t gfp_mask )

map user data to a request, for passthrough requests

Parameters
  struct request_queue *q
    request queue where request should be inserted
  struct request *rq
    request to map data to
  struct rq_map_data *map_data
    pointer to the rq_map_data holding pages (if necessary)
  const struct iov_iter *iter
    iovec iterator
  gfp_t gfp_mask
    memory allocation flags

Description
  Data will be mapped directly for zero copy I/O, if possible. Otherwise
a kernel bounce buffer is used.
