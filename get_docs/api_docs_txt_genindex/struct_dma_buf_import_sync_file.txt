Struct: struct dma_buf_import_sync_file

Insert a sync_file into a dma-buf

Members
  flags
    Read/write flags Must be DMA_BUF_SYNC_READ, DMA_BUF_SYNC_WRITE, or both. If DMA_BUF_SYNC_READ is set and DMA_BUF_SYNC_WRITE is not set,
this inserts the sync_file as a read-only fence.  Any subsequent
implicitly synchronized writes to this dma-buf will wait on this
fence but reads will not. If DMA_BUF_SYNC_WRITE is set, this inserts the sync_file as a
write fence.  All subsequent implicitly synchronized access to
this dma-buf will wait on this fence.
  fd
    Sync file descriptor
