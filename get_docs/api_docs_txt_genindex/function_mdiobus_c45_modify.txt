Function: int mdiobus_c45_modify ( struct mii_bus * bus , int addr , int devad , u32 regnum , u16 mask , u16 set )

Convenience function for modifying a given mdio device register

Parameters
  struct mii_bus *bus
    the mii_bus struct
  int addr
    the phy address
  int devad
    device address to read
  u32 regnum
    register number to write
  u16 mask
    bit mask of bits to clear
  u16 set
    bit mask of bits to set
