Function: netdev_features_t netdev_increment_features ( netdev_features_t all , netdev_features_t one , netdev_features_t mask )

increment feature set by one

Parameters
  netdev_features_t all
    current feature set
  netdev_features_t one
    new feature set
  netdev_features_t mask
    mask feature set

Description
  Computes a new feature set after adding a device with feature set one to the master device with current feature set all .  Will not
enable anything that is off in mask . Returns the new feature set.
