Function: int v4l2_subdev_set_routing ( struct v4l2_subdev * sd , struct v4l2_subdev_state * state , const struct v4l2_subdev_krouting * routing )

Set given routing to subdev state

Parameters
  struct v4l2_subdev *sd
    The subdevice
  struct v4l2_subdev_state *state
    The subdevice state
  const struct v4l2_subdev_krouting *routing
    Routing that will be copied to subdev state

Description
  This will release old routing table (if any) from the state, allocate
enough space for the given routing, and copy the routing.
