Function: void drm_dp_mst_get_port_malloc ( struct drm_dp_mst_port * port )

Increment the malloc refcount of an MST port

Parameters
  struct drm_dp_mst_port *port
    The struct drm_dp_mst_port to increment the malloc refcount of

Description
  Increments drm_dp_mst_port.malloc_kref . When drm_dp_mst_port.malloc_kref reaches 0, the memory allocation for port will be released and port may
no longer be used.
