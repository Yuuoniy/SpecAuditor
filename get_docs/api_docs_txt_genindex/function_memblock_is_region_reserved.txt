Function: bool __init_memblock memblock_is_region_reserved ( phys_addr_t base , phys_addr_t size )

check if a region intersects reserved memory

Parameters
  phys_addr_t base
    base of region to check
  phys_addr_t size
    size of region to check

Description
  Check if the region [ base , base + size ) intersects a reserved
memory block.
