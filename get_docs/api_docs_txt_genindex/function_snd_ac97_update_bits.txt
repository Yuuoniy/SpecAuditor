Function: int snd_ac97_update_bits ( struct snd_ac97 * ac97 , unsigned short reg , unsigned short mask , unsigned short value )

update the bits on the given register

Parameters
  struct snd_ac97 *ac97
    the ac97 instance
  unsigned short reg
    the register to change
  unsigned short mask
    the bit-mask to change
  unsigned short value
    the value to set

Description
  Updates the masked-bits on the given register only when the value
is changed.
