Function: size_t tty_insert_flip_string_flags ( struct tty_port * port , const u8 * chars , const u8 * flags , size_t size )

add characters to the tty buffer

Parameters
  struct tty_port *port
    tty port
  const u8 *chars
    characters
  const u8 *flags
    flag bytes
  size_t size
    size

Description
  Queue a series of bytes to the tty buffering. For each character the flags
array indicates the status of the character.
