Function: struct scatterlist * pci_p2pmem_alloc_sgl ( struct pci_dev * pdev , unsigned int * nents , u32 length )

allocate peer-to-peer DMA memory in a scatterlist

Parameters
  struct pci_dev *pdev
    the device to allocate memory from
  unsigned int *nents
    the number of SG entries in the list
  u32 length
    number of bytes to allocate
