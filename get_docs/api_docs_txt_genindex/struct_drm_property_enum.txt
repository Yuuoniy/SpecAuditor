Struct: struct drm_property_enum

symbolic values for enumerations

Members
  value
    numeric property value for this enum entry If the property has the type DRM_MODE_PROP_BITMASK , value stores a
bitshift, not a bitmask. In other words, the enum entry is enabled
if the bit number value is set in the property’s value. This enum entry has the bitmask 1 << value .
  head
    list of enum values , linked to drm_property.enum_list
  name
    symbolic name for the enum
