Function: struct drm_property * drm_property_create_enum ( struct drm_device * dev , u32 flags , const char * name , const struct drm_prop_enum_list * props , int num_values )

create a new enumeration property type

Parameters
  struct drm_device *dev
    drm device
  u32 flags
    flags specifying the property type
  const char *name
    name of the property
  const struct drm_prop_enum_list *props
    enumeration lists with property values
  int num_values
    number of pre-defined values

Description
  This creates a new generic drm property which can then be attached to a drm
object with drm_object_attach_property() . The returned property object must
be freed with drm_property_destroy() , which is done automatically when
calling drm_mode_config_cleanup() .
