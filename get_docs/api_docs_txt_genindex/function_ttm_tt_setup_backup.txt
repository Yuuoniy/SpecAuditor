Function: int ttm_tt_setup_backup ( struct ttm_tt * tt )

Allocate and assign a backup structure for a ttm_tt

Parameters
  struct ttm_tt *tt
    The ttm_tt for wich to allocate and assign a backup structure.

Description
  Assign a backup structure to be used for tt backup. This should
typically be done at bo creation, to avoid allocations at shrinking
time.
