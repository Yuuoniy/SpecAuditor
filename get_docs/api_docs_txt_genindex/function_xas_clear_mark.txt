Function: void xas_clear_mark ( const struct xa_state * xas , xa_mark_t mark )

Clears the mark on this entry and its parents.

Parameters
  const struct xa_state *xas
    XArray operation state.
  xa_mark_t mark
    Mark number.

Description
  Clears the specified mark on this entry, and walks back to the head
attempting to clear it on all the ancestor entries.  Does nothing if xas has not been walked to an entry, or is in an error state.
