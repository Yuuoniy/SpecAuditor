Function: int mdiobus_c45_read ( struct mii_bus * bus , int addr , int devad , u32 regnum )

Convenience function for reading a given MII mgmt register

Parameters
  struct mii_bus *bus
    the mii_bus struct
  int addr
    the phy address
  int devad
    device address to read
  u32 regnum
    register number to read
