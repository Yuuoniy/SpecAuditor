Function: struct srcu_ctr __percpu * srcu_read_lock_fast ( struct srcu_struct * ssp )

register a new reader for an SRCU-protected structure.

Parameters
  struct srcu_struct *ssp
    srcu_struct in which to register the new reader.

Description
  Enter an SRCU read-side critical section, but for a light-weight smp_mb() -free reader.  See srcu_read_lock() for more information.
