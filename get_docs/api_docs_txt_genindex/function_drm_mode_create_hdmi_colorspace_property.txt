Function: int drm_mode_create_hdmi_colorspace_property ( struct drm_connector * connector , u32 supported_colorspaces )

create hdmi colorspace property

Parameters
  struct drm_connector *connector
    connector to create the Colorspace property on.
  u32 supported_colorspaces
    bitmap of supported color spaces

Description
  Called by a driver the first time it’s needed, must be attached to desired
HDMI connectors.
