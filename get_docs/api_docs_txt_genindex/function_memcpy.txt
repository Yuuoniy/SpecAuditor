Function: void * memcpy ( void * dest , const void * src , size_t count )

Copy one area of memory to another

Parameters
  void *dest
    Where to copy to
  const void *src
    Where to copy from
  size_t count
    The size of the area.

Description
  You should not use this function to access IO space, use memcpy_toio() or memcpy_fromio() instead.
