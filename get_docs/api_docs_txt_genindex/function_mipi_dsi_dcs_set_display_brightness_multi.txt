Function: void mipi_dsi_dcs_set_display_brightness_multi ( struct mipi_dsi_multi_context * ctx , u16 brightness )

sets the brightness value of the display

Parameters
  struct mipi_dsi_multi_context *ctx
    Context for multiple DSI transactions
  u16 brightness
    brightness value

Description
  Like mipi_dsi_dcs_set_display_brightness() but deals with errors in a way that
makes it convenient to make several calls in a row.
