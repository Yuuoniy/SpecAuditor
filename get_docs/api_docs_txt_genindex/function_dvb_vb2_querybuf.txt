Function: int dvb_vb2_querybuf ( struct dvb_vb2_ctx * ctx , struct dmx_buffer * b )

Wrapper to vb2_core_querybuf() for Digital TV buffer handling.

Parameters
  struct dvb_vb2_ctx *ctx
    control struct for VB2 handler
  struct dmx_buffer *b
    struct dmx_buffer passed from userspace in
order to handle DMX_QUERYBUF .
