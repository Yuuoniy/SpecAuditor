Function: void xdr_init_decode ( struct xdr_stream * xdr , struct xdr_buf * buf , __be32 * p , struct rpc_rqst * rqst )

Initialize an xdr_stream for decoding data.

Parameters
  struct xdr_stream *xdr
    pointer to xdr_stream struct
  struct xdr_buf *buf
    pointer to XDR buffer from which to decode data
  __be32 *p
    current pointer inside XDR buffer
  struct rpc_rqst *rqst
    pointer to controlling rpc_rqst, for debugging
