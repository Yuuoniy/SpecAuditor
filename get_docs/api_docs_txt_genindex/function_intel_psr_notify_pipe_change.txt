Function: void intel_psr_notify_pipe_change ( struct intel_atomic_state * state , struct intel_crtc * crtc , bool enable )

Notify PSR about enable/disable of a pipe

Parameters
  struct intel_atomic_state *state
    intel atomic state
  struct intel_crtc *crtc
    intel crtc
  bool enable
    enable/disable

Description
  This is targeted for underrun on idle PSR HW bug (Wa_16025596647) to apply
remove the workaround when pipe is getting enabled/disabled
