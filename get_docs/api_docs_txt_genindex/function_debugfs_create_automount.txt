Function: struct dentry * debugfs_create_automount ( const char * name , struct dentry * parent , debugfs_automount_t f , void * data )

create automount point in the debugfs filesystem

Parameters
  const char *name
    a pointer to a string containing the name of the file to create.
  struct dentry *parent
    a pointer to the parent dentry for this file.  This should be a
directory dentry if set.  If this parameter is NULL, then the
file will be created in the root of the debugfs filesystem.
  debugfs_automount_t f
    function to be called when pathname resolution steps on that one.
  void *data
    opaque argument to pass to f().

Description
  f should return what -> d_automount() would.
