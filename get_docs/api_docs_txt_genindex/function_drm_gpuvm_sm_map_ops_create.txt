Function: struct drm_gpuva_ops * drm_gpuvm_sm_map_ops_create ( struct drm_gpuvm * gpuvm , const struct drm_gpuvm_map_req * req )

creates the drm_gpuva_ops to split and merge

Parameters
  struct drm_gpuvm *gpuvm
    the drm_gpuvm representing the GPU VA space
  const struct drm_gpuvm_map_req *req
    map request arguments

Description
  This function creates a list of operations to perform splitting and merging
of existing mapping(s) with the newly requested one.
