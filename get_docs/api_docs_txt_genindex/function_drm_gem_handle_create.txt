Function: int drm_gem_handle_create ( struct drm_file * file_priv , struct drm_gem_object * obj , u32 * handlep )

create a gem handle for an object

Parameters
  struct drm_file *file_priv
    drm file-private structure to register the handle for
  struct drm_gem_object *obj
    object to register
  u32 *handlep
    pointer to return the created handle to the caller

Description
  Create a handle for this object. This adds a handle reference to the object,
which includes a regular reference count. Callers will likely want to
dereference the object afterwards.
