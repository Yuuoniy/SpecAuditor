Function: int pci_p2pdma_add_resource ( struct pci_dev * pdev , int bar , size_t size , u64 offset )

add memory for use as p2p memory

Parameters
  struct pci_dev *pdev
    the device to add the memory to
  int bar
    PCI BAR to add
  size_t size
    size of the memory to add, may be zero to use the whole BAR
  u64 offset
    offset into the PCI BAR

Description
  The memory will be given ZONE_DEVICE struct pages so that it may
be used with any DMA request.
