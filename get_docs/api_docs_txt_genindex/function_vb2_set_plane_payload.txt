Function: void vb2_set_plane_payload ( struct vb2_buffer * vb , unsigned int plane_no , unsigned long size )

Parameters
  struct vb2_buffer *vb
    pointer to struct vb2_buffer to which the plane in
question belongs to.

    plane number for which payload should be set.
