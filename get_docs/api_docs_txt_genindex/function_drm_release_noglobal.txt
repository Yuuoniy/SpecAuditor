Function: int drm_release_noglobal ( struct inode * inode , struct file * filp )

release method for DRM file

Parameters
  struct inode *inode
    device inode
  struct file *filp
    file pointer.

Description
  This function may be used by drivers as their file_operations.release method. It frees any resources associated with the open file prior to taking
the drm_global_mutex. If this is the last open file for the DRM device, it
then restores the active in-kernel DRM client.
