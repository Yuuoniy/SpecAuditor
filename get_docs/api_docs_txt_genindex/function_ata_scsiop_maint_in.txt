Function: unsigned int ata_scsiop_maint_in ( struct ata_device * dev , struct scsi_cmnd * cmd , u8 * rbuf )

Simulate a subset of MAINTENANCE_IN

Parameters
  struct ata_device *dev
    Target device.
  struct scsi_cmnd *cmd
    SCSI command of interest.
  u8 *rbuf
    Response buffer, to which simulated SCSI cmd output is sent.

Description
  Yields a subset to satisfy scsi_report_opcode()
