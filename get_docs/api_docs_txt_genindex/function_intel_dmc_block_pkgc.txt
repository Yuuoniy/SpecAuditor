Function: void intel_dmc_block_pkgc ( struct intel_display * display , enum pipe pipe , bool block )

block PKG C-state

Parameters

    display instance
  enum pipe pipe
    pipe which register use to block
  bool block
    block/unblock

Description

PIPEDMC_BLOCK_PKGC_SW_BLOCK_PKGC_ALWAYS bit in PIPEDMC_BLOCK_PKGC_SW register.
